# Architecture Decision Records

This directory records the main architectural decisions for Bulk Image Size Reducer.

| ADR | Status | Decision |
| --- | --- | --- |
| [0001](0001-browser-only-processing.md) | Accepted | Process selected images entirely in the browser |
| [0002](0002-static-site-deployment.md) | Accepted | Keep the app deployable as a static site |
| [0003](0003-browser-canvas-encoding.md) | Accepted | Use browser canvas encoding for image output |
| [0004](0004-in-browser-zip-export.md) | Accepted | Generate ZIP downloads in the browser without runtime dependencies |
